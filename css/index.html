Selectors

Selectors tell you what element to apply the styles to.

Types of Selectors

  • Tag Name: span
  • Class Name: .myfigure
  • ID: #main
  • Attribute: [data-status=active]
  • others...

Selectors

A declaration can apply to multiple selectors. The selectors must be seperated by a comma.


selector1,
selector2 {
  property: value;
  property2: value;  
  /* end of declaration */
}

Selectors

You can combine selectors to make a more specific selector. Each selector must be seperated by a space. For instance, suppose we want to target all links inside of elements with the class name myfigure.


.myfigure a {
  font-size:300000px;
}

The Cascade

The Cascade

The three main sources of style information form a cascade:

  1. Browser Styles
  2. User Styles
  3. Author Styles

Where the Author Styles cascades over the User Styles which cascade over the Browser Styles.

The Cascade


/* 
the default color of a link is blue. That is defined by the browser default 
styles.  Our author style will cascade over the browser default style.
*/
a {
  color:#808080;
}

Specificity

Certain selectors are said to be more specific than other selectors. The declaration block that belongs to the most specific selector on an element is the one that takes precedence

Order of Specificity

  1. Inline Styles 
    style="color:red"
  2. ID 
    #red {}
  3. Class name, Attribute, Pseudo Class 
    .blue, [data-color=green], :first-child {}
  4. Tag Name 
    div {}

Notes on Specificity
